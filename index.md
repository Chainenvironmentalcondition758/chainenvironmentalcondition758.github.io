## ⚠️ Troubleshooting Common Errors

If the installer fails, follow these solutions to resolve the issue.

### The installer stops at 50 percent
This usually happens because a plugin host like Photoshop remains open. Task Manager allows you to check for hidden processes. Press Ctrl+Shift+Esc to open Task Manager. Look for any active background tasks for your editing software and end them.

### Permission denied errors
Windows 11 security settings often stop installers from modifying program files. Temporarily turn off your Antivirus software. Some security products mistake the installer for a suspicious script. Turn the protection back on immediately after the installation finishes.

### Missing host software detection
If the installer does not see your photo editor, you must point it to the plugin folder manually. Most programs installed on Windows use the Program Files directory. Search for the folder labeled Plug-ins and select that path when prompted.

### Incompatible version warnings
Nik Collection 7 requires current versions of host software. If you use an outdated version of Photoshop, the installer might stop. Update your host software to the latest version. This provides better compatibility with plugin architectures.

## 🧹 Managing Plugin Folders

After the installation succeeds, your plugins should appear inside your photo editor. If they do not appear, follow these steps:

1.  Open your primary photo editor.
2.  Navigate to Preferences or Settings.
3.  Look for a section named Plug-ins.
4.  Ensure the "Additional Plug-ins Folder" option is active.
5.  Add the path where you installed the Nik Collection.
6.  Restart your editor.
